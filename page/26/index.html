notepad++删除包含指定字符串的行（正则）

  • 注意： Notepad++的[全部替换]受[方向]约束，所以如果想“向下”全部替换，要把光标放到文首。
  • 如果留空行： 查找： ^.大师兄.$ 替换为：（空）
  • 如果不留空行： 查找： ^.大师兄.\r?\n 替换为：（空）

将字母大小写转换

  • 小写转大写：Ctrl + Shift + U
  • 大写转小写：Ctrl + U

  • Ctrl + Shift + X：将所选字符转为大写

  • Ctrl + Shift + R：全局查找
  • Ctrl + Shift + G：查找方法或属性的引用（就是查找在那些地方也用过）

  • Ctrl + Shift + Enter：在当前行插入空行

  • Alt + Shift + L：查看所有操作的快捷键
  • Ctrl + D：删除一行
  • Ctrl + 1：迅速定位错误处理
  • Ctrl + M：快速对当前视图最大化
  • Ctrl + N：新建文件

  • Ctrl + E：快速显示当前editor的下拉列表， ctrl+page down + ctrl+page up

  • Ctrl + O：在类中显示方法和属性的大纲
  • Ctrl + L ：快速定位当前编辑器的某一行
  • Ctrl + F6：定位到下一个编辑器
  • Ctrl + F ： 查找
  • Ctrl + /：代码注释/取消注释
  • Ctrl + T：快速显示当前类的继承结构
  • Ctrl + W：关闭当前编辑器
  • Alt + /：代码注释
  • Alt + ↑：向上移动选中行
  • Alt + ↓：向下移动选中行
  • Alt + →←：后退历史记录或前进历史记录
  • F3：快速定位光标位置的某个类，方法，属性（在哪定义的）
  • F4：显示类的继承关系
  • F5：单步跳入
  • F6：单步跳过
  • F7：单步跳出
  • F8：继续
  • Shift + enter：在当前行的下一行插入空行
