2.安装tampermonkey的百度云下载插件

  1. 点击浏览器右侧的【tampermonkey】的图标，弹出下拉框之后，选择【获取新脚本】
  2. 选择【GreasyFoek】点击（只有这一个可以访问，其他的都不好使）
  3. 在GreasyFoek的网页中搜索或者在【脚本列表】中找想要的脚本，例如：“百度网盘助手”等
  4. 安装完成之后，打开百度云网盘，在【tampermonkey】的图标上会有红色数字，点开看看有没有你的脚本
  5. 找到你要下载的资源，然后【选中】，点击导航栏的【下载助手】–> 【API下载】–> 【直接下载】(这个时候如果你安装了迅雷或者其他的下载软件就可以下载了，不过推荐使用Neat Download Manager 和 IDM )

3.Neat Download Manager下载及安装

Neat Download Manager主要用于mac（windows也可以使用），IDM主要用于windows
百度搜索Neat Download Manager，然后下载安装就可以，安装之后配置一下chrome的插件
