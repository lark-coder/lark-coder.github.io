《河岸》的阅读摘录

摘录

  • 有一次我清楚地听见母亲高亢的愤怒的声音传到了窗外，库文轩，坦白从宽，抗拒从严！这声音来自我家的房间，来自我的父亲母亲之间，让我感到滑稽，同时又感到恐怖。
  • 我记得很清楚，走过王六指家的舷板时，我还朝他女儿打了个轻松的响指，我没有表现出我的失落，反常的是我的后背，它还保持着惯性，微微弓起来，承接着一个虚拟的小女孩的身体，这很反常，分别一分钟，我的后背竟然开始思念起那个小女孩了。
  • 如果这是一个传奇，必须承认，传奇的开场有点像一幕闹剧，偏偏这闹剧是令人心酸的，向阳船队派了那么多人上岸，走了那么多冤枉路，费了那么大的周折，磨嘴皮子没用，骂娘动拳头没用，我的笔杆子也派不上用场，大家齐心协力，还是送不走一个小女孩，最后德盛又把慧仙驮回肩上，就像驮着一件沉重而神秘的礼品，一行人疲惫不堪地回到了船队。
  • 我无心描述幸福是什么样的，又不忍心玷污这个字眼，就深入浅出地说，幸福都是以后的事，等你找到妈妈，你就幸福了。
  • 向阳花的前途还用算？它的花盘只要朝着太阳，就有前途，什么时候花盘垂下来，对不准太阳，花就败了，就没有前途啦。
  • 我想象慧仙听见这些事的反应，她一定要说，你条件还挺高呢，那你喜欢什么样的姑娘呢？要不要我介绍一个给你？如果她这么问我，我该怎么说？这是我最大的秘密，说出这个秘密，需要精心准备的，要有技巧，我绝对不会说，我的条件不高，只要长得跟你一样的就行。
  • 为了慧仙，我人在船上，心在岸上。
  • 他们听不见我的命令，他们偏要和慧仙谈头发，我眼睛里的照相机还半遮半掩地对准慧仙，我耳朵里的留声机则采取罢工手段，愤怒地关上了。
  • 为什么每一次进理发店的时候我都满腔柔情，为什么每一次离开的时候我的心里会充满愤怒和怨恨呢？爱为什么会变成恨？我不甘心。我不甘心，所以我沿着灯蛾扑火的路线，一次一次地朝人民理发店走。
  • 院子里的枣树还在，一片枣树叶子落在我头上，我甩了甩头，树叶掉到了我的肩上，这是树叶的善意，枣树叶子欢迎我。

读后感

  • 破庙中神易，破心中神难。
  • 物质的匮乏，精神的崩溃，好几代人心灵的创伤。
  • 通篇没有任何感情的流露，这才是最可怕的，沦为生活的奴隶。
  • 荒诞不经的背后是扭曲和压抑到极致的人性。
