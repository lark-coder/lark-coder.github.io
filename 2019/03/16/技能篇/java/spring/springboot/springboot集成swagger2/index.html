注意事项

  1. swagger访问地址：http://localhost:8080/swagger-ui.html
  2. swagger需要自己配置汉化
  3. 其他swagger配置需要在使用的时候或者有需求的时候再增加配置
  4. swagger的不同版本的来回替换会出现bug
  5. 返回的long类型的数据在swagger的返回结果种会出现bug，数据会发生失真
